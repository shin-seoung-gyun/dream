텍스트 표시
- 특별한 태그 없이도 <body></body>태그 안에 있으면 표시할수있다.
-단락을 표시하지 않으면 모든 텍스트가 연결되어 하나의 긴 줄로 표시된다.

p (단락)
-단락이란 하나하나의 짧은 이야기의 토막
- 단락 전 후 에 빈 줄이 추가 된다

br태그
- 강제 줄바꿈 태그

<pre> 태그
- 프로그래머가 입력한 그대로 표시하는 태그

헤딩
-웹 페이지의 머리기사(headline)

주석
-코드를 설명하는글

<hr> 태그
- 수평선을 그리는 태그

특수문자
-&nbsp; -한개의 공백을 표현하는 특수문자

리스트
-번호있는 리스트 : ol태그 사용후 li태그 사용
-번호 없는 리스트 : ul태그 사용후 li태그 사용

정의 리스트
- 항목들과 함께 항목들의 정의(설명)가 표시되는 리스트
<dl>
	<dt>항목</dt>
	<dd>항목설명</dd>
</dl>

하이퍼 링크(링크)
-다른문서로 점프할수 있는 단어나 이미지

target - 속성
- 속성은 각 링크가 클릭되었을 때, 새로운 페이지가 어디에서 열리
는 지를 지정

id - 속성
- 점프하거나 해당 태그를 특정하기위해 이름을 만들어주는 속성

img 태그
- 웹 페이지에 이미지를 사용할때 사용하는 태그
<img src="이미지 위치" width = "너비" height="높이"/>
속성 width, height는 이미지의 크기를 나타낸다.

alt속성 
- 브라우저가 어떤 이유로 이미지를 화면에 표시하지 못했을 경우에,
표시되는 대체 텍스트를 지정.
시각장애인이 그림을 '읽는'것에 도움을 주므로 꼭 넣어주는것이 좋음

테이블 태그
- 웹에서 표를 그리는 태그
<table border="1"> // 표에 선을 그리기위한 속성 
	<tr> //하나의 행을 뜻하는 태그
		<th></th> //표의 머릿말을 담당하는 태그
	</tr>
	<tr>
		<td></td> //표에서 하나의 데이터를 뜻하는 태그
	</tr>
</table>

테이블 행, 열 병합
<td colspan="병합할 열수"> //열을 병합하는 속성.
<td rowspan="병합할 행수"> // 행을 병합하는 속성.

테이블 캡션 태그
- 테이블 태그 안에 caption 태그를 사용하면 표 밖에 표의 이름을
만들어 줄수 있다
