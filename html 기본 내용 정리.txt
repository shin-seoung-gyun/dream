HTML = Hyper Text Markup Language
- 웹 페이지를 기술하기 위한 마크업(Markup)언어

마크업(Markup)언어
-텍스트에 태그를 붙여서 텍스트가 문서의 어디에 해당하는지를 
기술한것

웹 브라우저
- HTML문서를 읽어서 눈에 보이는 웹 페이지를 만든다.

요소 element
-시작태그와 종료태그로 이루어진 문서의 구성요서
-요소 = (시작 태그 + 콘텐츠 + 종료 태그)

속성
-속성은 요소에 대한 추가적인 정보를 제공
-속성은 항상 시작태그에 이름 = "값" 형태로 기술된다.

주석 comment
-Html 코드를 설명하는글
ex) <!--주석을 여기에 표시한다.-->

<!DOCTYPE>
-웹페이지에 사용된 Html의 종류와 버전을 지정
-HTML5
<!DOCTYPE html>

Html 맛보기
<html>
	<head>
		<title>Web Programming</title>
	</head>
	<body>
		<h1>Welcome to the Web Programming!</h1>
		<img src = "이미지주소">
		<p>
		언제든지 오셔서 질문이 있으시면 올려주세요!
		<em>여러분을 환영합니다</em>
		</p>
		<h2>내용</h2>
		<p>
		HTML5,CSS,Javascript,JQuery,SQL,JSP,...
		</p>
	</body>
</html>
