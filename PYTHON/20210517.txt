리눅스(우분투) 20.04.20버전 (필드에서는 18.~~버전을 사용)
데스크탑 우분투는 GUI를 제공
서버용 우분투는 터미널창으로 작동

윈도우 콘솔은 CMD, POWERSHELL이있음 CMD보다 POWERSHELL이 더 기능이 강력하고 
리눅스에 가까움.

윈도우 CMD 기본 명령어
dir = 현재위치 디렉토리(폴더) 확인
cd = change directory 의 약자 현재위치를 변경한다. 리눅스에서도 같은 용도로 사용됨
cd .. = 상위 디렉토리로 이동(리눅스에서도 같음)
//tab을 사용하면 어느정도 자동완성을 한다.
윈도우상에서 실행하는 터미널은 대소문자 구분을 하지 않는다.
리눅스에서는 대소문자 구분을 하므로 윈도우에서도 대소문자 구분을 하는것이 좋다.
(습관)

리눅스 터미널 기본 명령어
ls =현재 위치 디렉토리 확인 (윈도우에서는 dir사용)
ls -l = 디렉토리 자세히 보기
ls -a = 디렉토리 숨김도 보기
ls -al = 모든 디렉토리 자세히 보기
clear = 터미널창 클리어
pwd = 현재 위치 확인
/ = 최상위 디렉토리인 root를 뜻함
~ = 현재 사용자의 홈 디렉토리
mkdir = make directory의 약자 디렉토리(폴더)만들기
vi = 리눅스에서 사용하는 편집기 //잘 안씀 빠져나오는 명령어 q!
nano = 리눅스에서 사용하는 텍스트 편집기 익숙하다
cat = 텍스트 파일 출력하는 명령어
head = 위에서 10줄 출력
tail = 뒤에서 10줄 출력
cp 복사할파일 복사할위치 = 복사 명령어 copy약자
mv = 이동 명령어 사용법은 cp와 상동
rm 지울 파일 = 삭제 명렁어

ssh = 리눅스 서버 원격접속 프로그램
ssh 계정명@서버아이피 = 사용방법(접속할 서버는 돌고있어야함)
